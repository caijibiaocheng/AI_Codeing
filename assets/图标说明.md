# 图标准备说明

## Windows 打包需要的图标

在 `assets` 目录下需要准备以下图标文件：

### 1. Windows (.ico)
- **文件名**: `icon.ico`
- **尺寸**: 256x256 像素（推荐包含多个尺寸：16x16, 32x32, 48x48, 64x64, 128x128, 256x256）
- **格式**: ICO

### 2. macOS (.icns)
- **文件名**: `icon.icns`
- **尺寸**: 512x512 或 1024x1024 像素
- **格式**: ICNS

### 3. Linux (.png)
- **文件名**: `icon.png`
- **尺寸**: 512x512 像素
- **格式**: PNG

## 如何创建图标

### 方法1：在线工具
1. 访问 https://www.icoconverter.com/ 或 https://convertico.com/
2. 上传一张 512x512 或 1024x1024 的 PNG 图片
3. 转换为 .ico（Windows）和 .icns（macOS）格式
4. 下载并放入 `assets` 目录

### 方法2：使用现有图标
如果你暂时没有图标，可以使用占位符图标：
1. 从 https://icon-icons.com/ 或 https://www.flaticon.com/ 下载免费图标
2. 使用在线工具转换格式

### 方法3：使用 electron-icon-builder（推荐）
```bash
npm install -g electron-icon-builder

# 准备一张 1024x1024 的 PNG 图片，命名为 icon.png
# 然后运行：
electron-icon-builder --input=./icon.png --output=./assets --flatten
```

## 临时解决方案
如果你现在就想打包，可以先跳过图标配置：
在 package.json 的 build 配置中，暂时注释掉 icon 相关的行。

打包后的软件会使用 Electron 默认图标。
